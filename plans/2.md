# Implementation Plan: 完善节点编辑功能

[Overview]
修改 MindmapNode.tsx，让编辑模式下的 textarea 使用绝对定位脱离 flex 布局约束，实现节点外框大小固定不变，但 textarea 可以随内容变大并超出节点边框显示。

[Types]
无类型系统变更，仅修改现有样式和结构。

[Files]

## webview/src/components/MindmapNode.tsx
- 添加 padding 常量定义
- 修改 node 外层 style（position, overflow, zIndex, cursor, userSelect）
- 编辑模式 JSX 结构改为 absolute wrapper 包裹 textarea
- 修改 mouseDown/MouseMove/MouseUp 事件处理（编辑模式下禁用拖拽）

[Functions]
无函数变更，仅修改组件内部逻辑。

[Classes]
无类变更。

[Dependencies]
无依赖变更。

[Testing]
1. 修改后执行 `npm run build:webview` 验证编译
2. 按 F5 测试扩展功能
3. 验证以下场景：
   - 编辑时节点外框大小不变
   - textarea 随内容变大并超出节点边框
   - textarea 没有滚动条
   - 点击节点不会触发拖拽
   - 展开/折叠按钮正常显示
