# Implementation Plan: 统一快捷键和保存逻辑

[Overview]
统一 VS Code 扩展的快捷键和保存逻辑，解决 Webview 内部操作栈与 VS Code 文件保存机制之间的割裂问题。

[Types]
无类型系统变更，仅修改现有逻辑和配置文件。

[Files]

## package.json
- 添加 `texmind.save` 命令
- 添加 Ctrl+S 保存的 keybinding

## webview/src/hooks/useMindmapShortcuts.ts
- 新建文件：全局快捷键监听 hook
- 捕获 Ctrl+S/Z/Y 并调用 store 或 postMessage

## webview/src/store/vscodeBridge.ts
- 新建文件：VS Code 通信桥接
- 提供 notifyDocumentChanged 和 requestSave 函数

## webview/src/App.tsx
- 引入 useMindmapShortcuts hook
- 调用 useMindmapShortcuts()

## webview/src/store/mindmapStore.ts
- 修改 undo/redo 方法，设置 isDirty: true 和 relayoutVersion
- 修改 setDocument 时清空 operationManager 栈

## src/TexMindEditorProvider.ts
- 添加 'save' 消息处理
- 添加 'mindmap.saved' 回复

[Dependencies]
无依赖变更。

[Testing]
1. 修改后执行 `npm run build:webview` 验证编译
2. 按 F5 测试扩展功能
3. 验证 Ctrl+S 保存、Ctrl+Z/Y undo/redo

[Implementation Order]

1. [ ] 修改 package.json 添加 save 命令和 keybinding
2. [ ] 创建 webview/src/hooks/useMindmapShortcuts.ts
3. [ ] 创建 webview/src/store/vscodeBridge.ts
4. [ ] 修改 webview/src/App.tsx 引入快捷键 hook
5. [ ] 修改 mindmapStore.ts 的 undo/redo 设置 dirty
6. [ ] 修改 mindmapStore.ts 的 setDocument 清空栈
7. [ ] 修改 TexMindEditorProvider.ts 添加消息处理
8. [ ] 执行 build:webview 验证
