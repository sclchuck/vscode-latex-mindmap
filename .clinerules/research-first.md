## Brief overview
当遇到不确定的技术问题时，优先通过联网搜索查找已有的解决方案，而不是直接尝试或猜测。

## Problem solving approach
- 遇到不确定的 API、错误信息或技术方案时，先联网搜索
- 搜索 Microsoft VS Code 官方文档、Stack Overflow、GitHub Issues 等资源
- 参考社区最佳实践后再实施解决方案
- 如果搜索无果，再尝试自己的解决方案并说明原因

## Debugging workflow
- 分析完日志并定位错误后，先联网搜索该错误问题
- 搜索该错误的相关讨论、解决方案
- 参考社区解决方案后再进行思考和实施
- 不要在未搜索的情况下直接尝试修复错误
